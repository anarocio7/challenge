Hi! This is a simple numbers game with HTML, CSS and Javascript.

Wanna play? These are the rules:

  • Choose a number between 1 and 100
  • Click on the Let's Guess button
  • If you guess, there will appear a message telling you that you've won
  • If you don't guess, you will have 4 more chances

Don't worry if you fail on the first try. Good Luck!
